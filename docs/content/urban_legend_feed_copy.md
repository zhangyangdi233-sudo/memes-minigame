# Urban Legend Feed Copy Notes

These notes guide short Chinese captions for the social masonry feed. The captions below are original game text, not copied platform posts.

## Tone Rules

1. Open with a question, request for verification, or implied invitation to compare experiences.
2. Use a concrete witness detail, then leave room for doubt.
3. Start from an ordinary place or device and introduce only one impossible change.
4. Keep the content type scannable through words such as record, screenshot, old post, rule, or live report.
5. End on a logical gap instead of naming a monster or explaining the cause.
